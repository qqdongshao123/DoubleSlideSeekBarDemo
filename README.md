## 使用方法

1. Maven仓库
在项目中引入:
```java
<dependency>
  <groupId>com.zds.view</groupId>
  <artifactId>doubleslideseekbar</artifactId>
  <version>1.0.0</version>
  <type>pom</type>
</dependency>
```
2. jeCenter
在项目的build.gradle文件中引入:
```java
  java
```
